Controls

Controls to Click

On the top of each cell
  • Top positions the cell to the top of the screens, starts voice unless mute
  • Next positions the next cell to the top of the screen, starts voice unless mute
  • Reload reloads the database, positions the cell to the top, stops voice
At the bottom of the sreen
  • Expand allows to display
    • the full text (all)
    • indicating only the common prayers and invocations (psalms)
    • indicating even the psalms and lections (nothing)
    • shoiwng only the main parts (skeleton)
    The last two options is for learning the structure only, always mute
  • Version allows to select one of the historical versions, see details in versions page
  • Mode allows to select the seasonal slant, onle the office for the season or saint.

    The seasonal is what I like: regular offices with the seasonal Capitula and responsory breves also preces and suffragia;
    Season option is allowed for Advent and Lent according to the 1960 Rubrics
  • Kalendar displays the calendar page to select an other date
  • Column2 selects the laguage for the second colum
  • Votive allows to chose some votive offices, or brings back the office for today
See the description of the tabs on the last line in help page

Keys to press

Control and numeric keys (in horas mode only)
  • ENTER positions the next page to the top, starts voice
  • digit 1-9 positions the indexed cell to the top (see index number at the top right of the second or only column)
  • F1 - F4 the decial digit part for the above number
  • F5 or H holds the voice, restart with R
  • F6 or R restarts after Hold
  • F7 or F closes (finish) popup, hours page or the programlet
  • F8 or D calls Editor
  • F9 positions to the end of the page, finishes the our if voice is set to continuous
  • 0 positions to the top of the page
  • Down arrow moves the page one line up
  • Insert moves the page one line up, the incremental motion is permanent for the future positionings
  • Up arrow moves the page one line down
  • Delete moves the page one line down, , the incremental motion is permanent for the future positionings
  • Page up moves the page half page up
  • Page down moves the page half page down
Alphabetic keys (case insensitive)
  • A calls Aperi Domine prayer for the behinning of the hour
  • C calls Completorium
  • D or F8 calls Editor
  • E calls Sancrosanctae prayer for the End of the office
  • F or F7 closes (finish) popup, hours page or the programlet
  • H or F5 holds the voice, restart with R
  • I interrupts the office, restart with Top, Nex, Enter or keying the index number
  • L calls Laudes
  • M calls Matutinum
  • N calls Nona
  • P calls Prima
  • R or F6 restarts after Hold
  • S calls Sexta
  • T calls Tertia
  • V calls Vespera
