HTML generation getting started

This page describes the HTML generator version of Divinum Officium programlet

Click here to download the latest version with the databases. The downloaded file will be unzipped creating Hofficium folder, with officium.exe program Hhoras.ini file points to horas subfolder contaning the databases (text files editable by any text processor producing ASCII text files). officiumprog folder contains the full set of the simbol files, either to this and the other versions.

Please help.

 The programlet may be started by
  • Hofficium.exe program using Windows operation system
  • Hofficium.pl in windows system if perl 5.8x is installed, for chant variation
  • LHofficium.pl in non windows systems, if perl 5.8x with Tk implement is installed
    The chant refences are omitted from this version.
    This version could be converted to exacutable by approriate program compatible with the give system
The programlet generates plain html files, into the 
$outdir/$year/
folder system, either one file for each day, or a master file and 8 files, one for each hour int $month-$day subfolder.

The programlet comes up with the page:



  • Click on the option widgets to select the appropriate variations for the generated files,

  • click on Options button to select some options and colors etc also for the generated files, and after clicking Chant tab to decide wether you want the images of the notes and wav files for the psalms and for some hymns or not

  • enter the from date and days to be generated into output folder

  • and finally click Generate offices button to perform the generation.

Maximum 366 folders may be generated at one time, this could overlap the current year.

Browse command calls your browser to be able to browse the generated files.

Hints:
  • You may select single language output by selecting the same language either as language for the second column and in Option dialog table as language for the first column.

  • Set full white backround by setting framecolor to 'white', and setting border = 0;

  • Bluish background will be generated by setting framecolor to "#ddddff", and unchecking white background box.
  • Turn of the accent by uncheckimh 'Options' 'Parameter' 'Accented' box

  • For Nook or Kindle set one file box checked, chant mute and notes off, Options Parameters Cell max chars input = is the maximum number of characters in a cell. (depends of the screensize and the font size, 300 for Nook medium fontsize, 200 for Kindle. Set 10000 or higher for keeping the original table cell system).

    For kindle set the number of dots to Extracolumn, larger number for smaller letter size. It seems to work, especially if you set the used language as first column. Extracolumn is zero for Nook
    Use Calibre freeware to convert 'EPUB' (Nook) or 'MOBI' (Kindle) format.
